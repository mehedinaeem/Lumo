# Speech-to-Text (STT) Models

This directory contains the STT models used by VOSK and the Whisper processor. 
Model directories are excluded from GitHub to keep the repository small.

## Expected Models
Download and extract the following folders here:
1. `vosk-model-small-en-us-0.15/`: English VOSK model. Download from [Alphacephei](https://alphacephei.com/vosk/models).
2. `BanglaASR/`: Fine-tuned Whisper model for Bangla. Download from Hugging Face: `Rakib/BanglaASR`.
