# Local LLM Models

This directory contains the large generative language models used for offline inference. 
Due to their size, the `.gguf` and `.bin` files are ignored by `.gitignore` and not pushed to GitHub.

## Expected Models
Download the following models and place them in this directory:
- `orca-mini-3b-gguf2-q4_0.gguf` (~1.9 GB)
- Find it at: [Hugging Face](https://huggingface.co/psmathur/orca_mini_3b) or via GPT4All.
